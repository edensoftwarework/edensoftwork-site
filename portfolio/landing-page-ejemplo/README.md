# Landing Page - Panadería Deliciosa

Una página web estática (landing page) para promocionar una panadería ficticia. Demuestra habilidades en diseño web responsivo y UX simple.

## Características
- Diseño responsivo con CSS Grid y Flexbox.
- Navegación suave con JavaScript.
- Secciones: Hero, Productos organizados por 8 categorías (Panes, Facturas, Tortas, Tartas, Galletas, Sandwiches, Cafés, Tés), Contacto.
- Catálogo extenso con 64 productos, imágenes únicas, descripciones y precios.
- Optimizada para conversión (llamado a acción en productos).

## Tecnologías
- HTML5
- CSS3
- JavaScript (básico)

## Cómo ejecutar
Abre `index.html` en un navegador web.

Este proyecto es parte del portfolio de EDEN SoftWork, mostrando capacidad para crear sitios web atractivos y funcionales con catálogos de productos organizados.
